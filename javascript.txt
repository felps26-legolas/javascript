MANIPULÇÃO DO DOM 
DOM é a estrutura em arvore que representa a página HTML

SELECIONANDO ELEMENTOS DO DOM

.document.getelementById: seleciona um elemento por seu ID
.document.queryselector:  seleciona o primeiro elemento que corresponde a um seletor CSS
.document.queryselector: seleciona todos os elementos que correspondem a um seletor CSS

ex:
let titulo = document.getelementbyid("titulo");

let paragrafo = document.queryselector(".paragrafo");

let botoes = document.queryselector("button");

EVENTOS 
permitem que voce execute codigo quando o usuario interage com a página,como clicar em um 
botão , pressionar uma tecla etc...

ex:  let botão = document.getelementbyid("meu botão");

     botão.addventlistener ("click", function(){alert("botão clicado!");});

EVENTOS COMUNS 

.CLICK: quando um elemento e clicado
.MOUSEROVER: quando o mouse passa sobre um elemento
