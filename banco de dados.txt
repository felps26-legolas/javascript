BANCO DE DADOS 

EVOLUÇÃO HISTÓRICA 

.Começou na década de 1960 com o desenvolvimento de banco de dados hierarquivos pela IBM

.em 1970 surgiram modelos de dados relacionais , que são base para muitos sistemas de banco
de dados

.em 1980, o SQL tornou-se padrão mundial para modelos relacionais

.em 1990, surgiram conceitos como data mining e data warehousing

.no ano 2000 , destacam-se tecnologias como  XML, redes sociais, computação em nuvem, etc.

O QUE É UM BANCO DE DADOS 

.Um banco de dados é uma coleção organizada de dados , podendo variar de simples arquivos de 
texto a complexos sistemas de gerenciamento de dados relacionais

.são utilizados em diversas áreas, como listas de telefones , sistemas de faturamento, redes
sociais, entre outros 

O QUE É UM SISTEMA GERENCIADOR DE BANCO DE DADOS (SGBD)

.SGBD é um conjunto de programas que permitem controlar o acesso, manipular dados e gerar
relatórios

.existem 4 modelos principais de SGBD:

1-HIERÁRQUICO: organizadados em uma estrutura de árvore, com relações um-para-muitos

2-REDE: permite relações complexas, com registros-filhos possuindo varios pais

3-RELACIONAL: organiza dados em tabelas (ou relações), com uma linguagem de consulta poderosa
(SQL)

4-ORIENTADO A OBJETOS: Combina dados e suas relações em uma única estrutura chamada objeto,
utilizando conceitos como classes e herança

A IMPORTANCIA DO BANCO DE DADOS

.Banco de dados são fundamentais para organizanização e armazenamento de informações dentro
de uma empresa 

.eles suportam a tomada de decisões estratégicas , através de sistemas que permitem a análise
e visualização de dados relevantes.

RESUMO

.A evolução dos bancos de dados reflete uma adaptação contínua ás  necessidades de armazenamento
e manipulação de dados

.desde os modelos hierarquicos e de rede até os sistemas relacionais e orientados a objetos
, cada tipo de SGBD oferece vantagens específicas para diferentes aplicações

.os bancos de dados são ferramentas essenciais para a gestão eficaz de informações,apoiando
a inovação e o sucesso empresarial.
