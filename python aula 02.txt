LINGUAGEM PYTHON

-> Simplicidade , compatibilidade , funciona na maioria dos sistemas opercionais

->capacidade é a capacidade de auxiliar outra linguagens 

->é considerada uma linguagem interpretada , orientada a objetos, de alto nível e com 
semnatica dinâmica 

CARACTERISTICAS DA LINGUAGEM PYTHON

->Linguagem de alto nível 
->Orientada a objetos
->Tipagem dinamica e forte
->Fácil aprendizagem
->Comunidade ativa
->Muitas bibliotecas/ recursos
->linguagem interpretada

O QUE É LINGUAGEM INTERPRETADA -> Linguagem de programação na qual o códogp é executado por
um programa de computador chamado interpretador

->Em seguida é executado pelo sistema opercaional ou processador

POR QUE É A MELHOR PARA INICIANTES?

FACILIDADE ->É uma linguagem orientada a objetos e tem a simplicidade como sua maior característica

SIMPLICIDADE-> O python não requer conhecimento de outras linguagens para que possa ser usado

->A lógica pode ser aplicada direto no código , o que faz seu aprendizado seja simples e rápido

->É uma linguagem de fonte live e aberta , com uma grande comunidade de apoio para os desenvolvedores

ROBUSTEZ ->Pode ser aplicado em projetos robustos 

VERSÕES DO PYTHON -> PYTHON 2 -> Foi padrão da linguagem por muito tempo

->Recebeu atualizações de segurança até 2020, quando seu suporte foi descontinuado

-> Use python 2 só quando estiver trabalhando com um software que ainda não foi
migrado para python 3  ou caso precise manter algum sistema legado

PYTHON 3 -> Introduziu algumas mudanças que quebraram a compatibilidade com a versão anterior,
o que criou a necessidade de se manter duas verões da linguagem

-> Está constantemente evoluindo e recebendo novas funcionalidades, que não estavam na versão
anterior

INTEGRATED DEVELOPMENT ENVIRONMENT (IDE)

->Ambiente de desenvolvimento integrado , é um software que combina ferramentas comuns para
desenvolvimento em uma única interface gráfica 

->facilita o desenvolvimento de aplicativos 

FACILIDADE NO DESENVOLVIMENTO COM IDE -> A utilização de uma IDE torna o desenvolvimento mais
fácil e agradavel

->Com um aspecto mordeno, o que vai facilitar no desenvolvimento das atividades

PYCHARM ->É uma IDE especifica para a linguagem PYTHON 

PYCHARM POSSUÍ -> Diversos recursos extremamente úteis , que facilitam as tarefas em desenvolvimento
