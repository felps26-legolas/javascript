ESTRUTURA DE DADOS LINEARES 05 

INTRODUÇÃO

.Os algoritmos de classificação em tempo linear não comparam todos os elementos no arranjo

.Algoritmos como bubble sort e insertion sort comparam repetidamente elementos , resultando em muitas comparações e maior tempo de execução

.A unidade avaliará algoritmos que ordenam um arranjo com um número de instruções proporcional a n , com algumas restrições

ORDENAÇÃO POR CONTAGEM (COUNTING SORT)

.Executa um número de instruções proporcional a N , exigindo que a entrada seja um arranjo de inteiros maiores ou iguais a 0 e menores ou
iguais a K

.O algoritmo utiliza um arranjo auxiliar C para armazenar as quantidades de cada valor observado no arranjo A, permitindo a ordenação
sem comparações diretas 

.O processo envolve a contagem das ocorrencias de cada elemento , a acumulação dessas contagens e a inserção dos elementos ordenados em 
um novo arranjo B

EXECUÇÃO DO COUTING SORT

.O algoritmo percorre o arranjo A do fim para o inicio , utilizando os valores como índice no arranjo C para determinar a posição de inserção
no arranjo B

.Após inserir um valor em B , o valor acumulado em C é decrementado para garantir que a próxima ocorrencia do mesmo valor
