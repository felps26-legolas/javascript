SISTEMAS OPERACIONAIS 

GERENCIADOR DE ENTRADA E SAÍDA

O QUE É?

.É uma das funções principais do sistema operacional

.É responsavel por controlar a  comunicação entre o computador (hardware) e o os dispositivos de entrada e saída como:

.TECLADO ->(ENTRADA)
.MOUSE ->(ENTRADA)
.MONITOR ->(SAÍDA)
.IMPRESSORA ->(SAÍDA)
.HDS ->(ENTRADA E SAÍDA)
.PENDRIVES ->(ENTRADA E SAÍDA)

COMO FUNCIONA?

.O usuario executa um comando (como imprimir um documento)

.O sistema operacional interpreta este comando e envia para o hardware correspondente

.O dispositivo realiza a ação (exibir no monitor , imprimir , etc )

PRINCIPAIS FUNÇÕES

.CONTROLE -> O SO controla como os dispositivos de E/S funcionam , mesmo que opere de forma diferente

.GERENCIAMENTO DE CONCORRENCIA -> Vários usuarios ou programas podem usar dispositivos ao  mesmo tempo

.ARMAZENAMENTO E ACESSO AOS DADOS -> O SO garante que os dados sejam guardados corretamente e evite conflitos 

.VELOCIDADE DE OPERAÇÃO -> Dispositivos tem velocidades diferentes , e o SO precisa lidar com isso

.CAMADAS DE CONTROLE -> O SO pode ter CAMADAS , cada uma com uma função especifica para controlar 

DRIVERS 

.São pequenos programas que permitem que o SO se comunique com o hardware 

CONTROLADORAS 

.São responsáveis por interligar o sistema operacional ao hardware

ex: A controladora de HD permite que o SO leia e escreva no disco

.Mesmo que troque o HD , se tiver uma controladora compativel , o sistema não é afetado

BARRAMENTO 

.É um canal de comunicação que transporta os dados entre os dispositivos e o processador 

.Ajuda a manter a comunicação mesmo como diferentes velocidades e tipos de dispositivos 

BIBLIOTECAS E SOFTWARES 

.O usuario pode interagir com dispositivos através de softwares e bibliotecas do sistema , como comandos em C , python etc
