ESTRUTURA BÁSICA DA LINGUAGEM PYTHON, DECLARAÇÃO DE VARIAVEIS E TIPODE DE DADOS

COMANDO PRINT() -> É utilizado quando desejamos imprimir alguma mensagem de texto na tela

SINTAX DA FUNÇÃO PRINT() -> Requer que se utilize os parenteses com o texto que será exibido

->Esse texto deve ser escrito,também, entre aspas, podendo ser aspas simples 'ou dupla'

CASE SENSITIVE -> A linguagem python é case sensitive

->Significa que ela faz distinção entre letras maiúsculas e minúsculas

-> É importante estar atento à escrita dos comandos

-> Ao executar um programa com erro de sintaxe, o resultado não será exibido e os erros
são apontados (console é uma maneira rápida de executar comandos no python)

VARIÁVEIS -> Ao iniciarmos uma variavel e atribuimos um valor à ela

->Na linguagem de programação python, as variaveis armazenam endereços de memória e não
valores

->É correto afirmar que o conceito de variavel em python é representado sempre por um objeto
e toda váriavel é uma referencia 

DECLARAÇÃO DE VÁRIAVEIS 

->Em python não é necessário declarar as váriaveis no inicio do código 

->isso não pode ser feito a qualquer momento, durante a codificação

->Sobre o tipo de variavel, não é necessario defini-lo em sua declaração

->A linguagem interpreta o tipo de dado que está sendo armazenado

->Caso for necessário forçar o reconhecimento de algum tipo especifico, basa realizar as conversões
necessárias

TIPOS DE VÁRIAVEIS -> Diferente de outras linguagens, uma váriavel em python tem um tipo fixo,
apenas o tipo do conteudo

EXEMPLO -> Ao atribuir o valor 10 na váriavel X, ela passará a ser reconhecida com um tipo
inteiro

CIDADE="Santos"

->Atribuindo o valor 'Santos' na variavel cidade, ela passará a ser reconhecida com um tipo 
string

->Tambem é possivel identificar os tipode básicos que são : inteiro(int) , real(float) ,
lógico/booleano (bool) e string (str)

TIPODS DE DADOS 

INTEIRO(INT) -> São aquelas que armazenam números inteiros 1,2,3 e assim por diante

REAL(FLOAT) -> Representa números reais e que possuem sinal de expoente (e ou E)

BOOLEANO(BOOL) -> Representam valores booleanos que são verdadeiro (true) e falso(false)

STRING(STR) -> Representa qualquer conjunto de caracteres, desde que estejam entre aspas

CONCATENAÇÃO DE DADOS NA SAÍDA PADRAO -> No momento de exibição podemos concatenar(unir)
várias informações em apenas uma linha

->Para isso utilizamos  a vírgula (,) para separar textos e as váriaveis

->Também podemos concatenar as informações na linguagem python utilizando o sinal de soma(+)

->Neste caso, temos de converter os valores que não são string para o tipo string , para
isso utilizamos o comando (str) antes da impressão da variavel

TABELA MASCÁRA DE FORMATAÇÃO

MÁSCARA ->%d ou %i | TIPO DE DADO -> int(inteiro) |DESCRIÇÃO-> Exibe um valor

MÁSCARA ->%f | TIPO DE DADO ->Float ou Double |DESCRIÇÃO ->Exibe um valor decimal

MÁSCARA ->%id | TIPO DE DADO ->Long int |DESCRIÇÃO ->Exibe um número inteiro

MÁSCARA ->%e ou %E | TIPO DE DADO ->Float e double |DESCRIÇÃO ->Exibe um número exponecial
                                                                (número cientifico)

MÁSCARA ->%c |TIPO DE DADO -> Char(caractere) |DESCRIÇÃO ->Exibe um caractere

MÁSCARA ->%o |TIPO DE DADO -> Int | DESCRIÇÃO ->Exibe um número inteiro em formato octal

MÁSCARA ->%x ou %X |TIPO DE DADO -> Int |DESCRIÇÃO ->Exibe um número inteiro em formato hexadecimal

MÁSCARA ->%s |TIPO DE DADO -> Char |DESCRIÇÃO ->Exibe uma cadeia de caracteres(string)

MÁSCARA ->%r |TIPO DE DADO -> Boolean |Exibe true ou false(verdadeiro ou falso)

ENTRADA DE DADOS ->Para que se possa ter uma interação do usuario tornando o programa dinamico

->Temos que trabalhar com a função de entrada de dados, que no python é conhecida como : input

ENTRADA DE DADOS NÃO STRING ->Quando utilizamos a entrada de dados por meio de um input 

->Essa será considerada do tipo string , caso seja necessário realizar a entrada de valores
numéricos, temos de converter o tipo de dado ,de acordo com o tipo que desejamos armazenar
na váriavel 

SEQUENCIA DE ESCAPES -> Utilizamos a sequencia de escapes para auxiliar na formatação da
exibição dos dados

->Para aplica-los, utilizamos uma combinação de caracteres associados a uma contra barra(\)

SEQUENCIA ->\n |DESCRIÇÃO ->Insere uma quebra de linha
            \t | -> Insere tabulação horizontal
            \v | -> Insere tabulação vertical
            \r | -> Equivalente ao efeito da tecla Enter
            \´ | -> Aspas simples
            \" | -> Aspas duplas
            \\ | -> Insere uma barra invertida (blackslash) 
            \a | -> Chamado de ASC bell ou beep do sistema , se houver suporte, aciona um bipe
            \b | -> Aciona o backspace, ou seja, apaga o caractere anterior
            \f | -> Insere uma quebra de página 
            \u | -> Insere um caractere UNICODE, Deve acompanhar um código com 4 números
